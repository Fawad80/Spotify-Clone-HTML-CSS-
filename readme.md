🎵 Spotify Clone
A responsive frontend web application that replicates the UI of the popular music streaming platform, Spotify. This project focuses on clean layout design using HTML and CSS.

✨ Features
Responsive Sidebar: Navigation for Home, Search, and Your Library.

Music Player UI: A functional-looking bottom player bar with play/pause icons and a progress bar.

Album Grid: A clean layout showing various playlists and album covers.

Hover Effects: Interactive buttons and card hover states to mimic the real Spotify experience.

Fully Responsive: Designed to look good on both desktop and mobile screens.

🛠️ Tech Stack
HTML5: For the structural layout.

CSS3: For custom styling, Flexbox, and CSS Grid.

FontAwesome: For high-quality music and navigation icons.
